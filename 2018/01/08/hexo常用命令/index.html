hexo常用命令

发表于 2018-01-08

很久都没有再写博客，主要还是因为在找到新工作之后没有时间···[挥汗]

正是因为很久没有写，经常用的一些命令都忘得差不多了，每次都要去百度，这次就做个总结；

hexo new my_first_blog = hexo n my_first_blog # 新建文章

hexo p = hexo publish # 发布文章

hexo g = hexo genarate # 生成文章

hexo s = hexo server # 启动服务预览

hexo d = hexo deploy # 部署文章

发布文章的时候，先找到“hexo blog”所在的本地文件夹，右键选择“Git bash here”，进入命令行界面；新建好文件之后，在“\source\_posts”下面就能找到刚才生成的文章，就可以使用编辑器进行编辑，推荐 Typora.

文章编辑结束以后，就可以准备发布了，可以使用下面的命令直接完成：

hexo generate –deploy = hexo deploy -g

hexo deploy –generate = hexo server -g

两个都可以,推荐第一个；
